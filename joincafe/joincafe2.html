홍길동
  • 접속 시 option 설정(옵션 페이지)을 해논 상태라면 해당 옵션의 설정과 동일하게 "금액"과 "조건"에 표시가 되어 있고, 우측의 On-Off 버튼은 On상태로 있는다.
  • Off로 변경 시 모든 선택항목은 "상관없어요"로 변경된다.
  • On으로 변경 시 option 상태와 동일하게 선택상태로 바뀐다.
  • On 상태에서 항목 중 하나라도 선택항목을 변경하면, Off상태로 바뀐다.
  • 전송하기 버튼을 누르면 현재 선택한 항목으로 서버에 전송한다.
    • 항목에 해당하는 사항으로 가맹점에 푸쉬하는 조건은 DB서버 페이지에서 확인 가능하다.
